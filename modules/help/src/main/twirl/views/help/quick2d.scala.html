  • Select database for MSA generation
    Select the database that should be used for MSA generation. You can select between: UniRef90, UniRef90, nr90, and nr70
  • Maximal no. of MSA generation steps
    The maximal number of PSI-BLAST for generating the MSA.
  • E-value incl. threshold for MSA generation
    Alignments are extended using the jump-start option of PSI-BLAST. Only include hits with E-value below this threshold into the resulting multiple sequence alignment.
