PhyML is a software that estimates maximum likelihood phylogenies from alignments of amino acid sequences.

  • Number of replicates
    Number of resampled datasets asked by SEQBOOT.

    Amino acid substitution model that is used for the computation of the distance matrix by PROTDIST (Please have a look at the link for a more detailed description).
