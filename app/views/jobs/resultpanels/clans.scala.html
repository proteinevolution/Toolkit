Download the zipped CLANS file (@{jobID}.clans.zip) generated for your sequences and unzip it.

Download the CLANS application here: clans.jar and visualize your unzipped CLANS file (@{jobID}.clans) using the 'Load Run' option in the 'File' menu.

Please make sure you have Java Runtime Environment (JRE) installed.

If your input contains large number of sequences, please consider running CLANS on the command line:
java -Xmx4G -jar clans.jar -load @{jobID}.clans

If you face problems loading the CLANS file, please increase the maximum Java heap size (-Xmx).
