Quick2D gives you an overview of secondary structure features like alpha-helices, extended beta-sheets, coiled coils, transmembrane helices and disorder regions.
  • Input
    You have the option to enter or upload either a single protein sequence or a multiple sequence alignment (MSA) in FASTA or CLUSTAL format.

    Single query sequence:

    You can enter a single query sequence in FASTA format (with or without name line):

    >gi|532319|pir|TVFV2E|TVFV2E envelope protein
    ELRLRYCAPAGFALLKCNDADYDGFKTNCSNVSVVHCTNLMNTTVTTGLLLNGSYSENRT
    QIWQKHRTSNDSALILLNKHYNLTVTCKRPGNKTVLPVTIMAGLVFHSQKYNLRLRQAWC

    MDWFLNYLNNLTVDADHNECKNTSGTKSGNKRAPGPCVQR

  • Select database for MSA generation
    Select the database that should be used for MSA generation. You can select between: UniRef90, UniRef90, nr90, and nr70
  • Maximal no. of MSA generation steps
    The maximal number of PSI-BLAST for generating the MSA.
  • E-value incl. threshold for MSA generation
    Alignments are extended using the jump-start option of PSI-BLAST. Only include hits with E-value below this threshold into the resulting multiple sequence alignment.
