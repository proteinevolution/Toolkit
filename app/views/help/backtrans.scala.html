The Backtranslator translates an amino acid sequence back into a possible DNA sequence.
  • Input
    Enter a single protein sequence in FASTA format.

    @views.html.help.common.singleseq()

  • Choose a genetic Code
    Select the correct genetic code for your organism. The genetic codes are identical to those available on the NCBI homepage.
  • Use codon usage table of
    Enter the species name (e.g. Homo sapiens, Escherichia coli) you want to use the codon table of. If this option is not used all possible codons will have the same probability. Check at the result page that the correct codon table has been used. You will see a message like: 
                        Generated using codon usage table of: Homo sapiens
