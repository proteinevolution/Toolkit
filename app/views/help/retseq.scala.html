This utility performs a sequence retrieval from the nre and nr database using a list of identifiers.
  • Input
    Enter a newline separated list of identifiers.
  • Retrieve identical sequence only once
    With this option the program filters identical identifiers and identical sequences, so that the result sequences are unique.
