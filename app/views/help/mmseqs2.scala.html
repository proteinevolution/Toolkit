MMseqs2 (Many-against-Many searching) is a software suite to search and cluster huge protein sequence sets. MMseqs2 can run 10000 times faster than BLAST. At 100 times its speed it achieves the same sensitivity. It can also perform profile searches with the same sensitivity as PSI-BLAST but at around 270 times its speed.
  • Input

    Enter multiple sequences in FASTA format. A sequence in FASTA format begins with a single-line description, followed by lines of sequence data. The description line is distinguished from the sequence data by a greater-than (">") symbol in the first column. You can enter up to 20,000 sequences. An example sequence in FASTA format is:

  • Minimum sequence identity
    List matches above this sequence identity (for clustering)
  • Minimum alignment coverage
    List matches above this fraction of aligned (covered) query and target residues
  • Clustering mode
    The 'normal' mode employs the slow (quadratic time), but sensitive 'cluster' pipeline offered by MMseqs2. The 'linclust' mode clusters sequences in linear time, but is less sensitive than the normal mode.
